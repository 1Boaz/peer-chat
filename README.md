# peer-chat
A p2p chat application written in go to send messages between computers.
###### Disclaimer this readme and project is in progress and some features may not be implemented or be missing from this readme

## 🌟 Highlights
- Written in go for memory efficiency and speed
- Will be updated as continued...

## Usage instructions
To start the cli application just run:
```shell
peer-chat -port <port number to use, default: 31415>
```
###### will be updated as changed

## Installation instructions
This CLI tool can be installed thorough 2 methods:
1. ### Quick install
    ```shell
    go install github.com/1Boaz/peer-chat@latest
    ```
2. ### Building from Source
    ```shell
    git clone https://github.com/1Boaz/peer-chat
    
    cd peer-chat
    
    go install
    ```
